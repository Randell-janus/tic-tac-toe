# Tic-tac-toe

Tic-tac-toe game using React.js and Tailwind CSS

- https://tic-tac-toe-randell-janus.vercel.app/

## Features

- Game modes: Single-player (default) & Two-player
- Toggle between different winning animations as you restart a round!
